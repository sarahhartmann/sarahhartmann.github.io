Hi,
I'm Sarah.

Who I am

I'm a 24-year-old Web Designer who is working on transitioning into UX. My short-term goal is to get a job in UX or Front-End Development. My long-term goal is to become a UX Manager. When I'm not working toward these goals, I enjoy getting brunch, going to concerts, and drinking craft beer.

I started coding before I even knew what it was (thanks Tumblr!) and after a slight detour in college, (Journalism Degree), I'm finally doing what I love!

What I do

For the past ten months, I've been creating websites using HTML5, CSS3, Wordpress, and Divi. I use hand-drawn paper wireframes, high-fidelity mockups in Adobe XD, and the designing in-browser technique to build sites.

I've done some backend setup as well, including creating temporary domains, navigating CPanel, and editing files.

How I do it

HTML

I use HTML5, making sure I pay attention to header tags and structure for SEO purposes.

CSS

I use CSS3 to style sites. I also use it to create animations using CSS transition and transform.

Adobe

I create using Photoshop, XD, InDesign, Illustrator, Premiere, After Effects, and Lightroom.

JavaScript

I'm still learning JS. My goal is to become a pro at JS animations so I can implement them in my sites.
